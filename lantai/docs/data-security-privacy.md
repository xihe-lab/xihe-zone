# 数据安全与隐私保护指南

**负责人：皋陶（gāo yáo）**

## 一、总则

1.  本指南依据《中华人民共和国个人信息保护法》《中华人民共和国数据安全法》《网络安全法》制定，规范羲和实验室数据收集、存储、使用、传输、销毁全流程，保障用户隐私与数据安全。

2.  核心原则：合法、正当、必要、最小范围，不收集无关数据，不泄露任何数据。

3.  适用范围：实验室官网、兰台文档中心、技术项目、智能体相关的数据处理活动，所有相关负责人需严格遵守。

## 二、数据收集合规要求

### （一）收集范围（仅收集以下必要数据，不额外收集）

- 用户自愿提交的咨询信息（如姓名、联系方式，仅用于回复咨询）；

- 官网、兰台页面的访问日志（仅用于技术优化、异常排查，不包含个人身份信息）；

- 文档下载记录（仅用于统计文档使用情况，不关联个人信息）。

### （二）收集规范

- 自愿原则：所有数据收集需获得用户明确同意，不强制、不诱导用户提交数据。

- 明确告知：收集前需清晰告知用户收集的数据类型、用途、保存期限，用户可自主选择是否提交。

- 最小必要：仅收集实现功能所需的最少数据，不收集与用途无关的信息（如无需收集身份证号、地址等）。

- 禁止强制：不将用户提交数据作为访问官网、浏览兰台文档的前提条件（公开内容无需登录即可访问）。

## 三、数据存储与保护

- 存储安全：用户数据需存储在安全服务器，采取加密、防火墙等防护措施，防止数据泄露、篡改、丢失。

- 保存期限：用户提交的咨询信息，在回复完毕后保存不超过30天，到期自动删除；访问日志、下载记录保存不超过6个月，到期清理。

- 权限控制：仅墨子（后端）、皋陶（gāo yáo）拥有数据访问权限，严格控制访问范围，禁止无关人员访问。

- 禁止备份：不擅自备份用户个人信息，确需备份的，需经皋陶（gāo yáo）审核，备份后采取同等安全保护措施。

## 四、数据使用与传输

- 使用限制：收集的数据仅用于明确告知的用途，不用于其他任何目的（如不用于营销、分享）。

- 匿名化处理：如需使用数据进行统计、分析，需对个人身份信息进行匿名化处理，无法识别具体用户。

- 传输安全：数据传输过程中需采取加密措施，禁止通过非安全渠道（如私人微信、邮箱）传输用户数据。

- 禁止泄露：不向任何第三方泄露、出售、出租用户数据，法律法规另有规定的除外。

## 五、数据销毁

- 到期销毁：按保存期限要求，到期后对数据进行彻底销毁（包括服务器存储、备份数据），确保无法恢复。

- 主动删除：用户提出删除个人数据请求的，需在24小时内核实身份，完成数据删除，并告知用户。

- 销毁记录：数据销毁需留存记录，注明销毁时间、数据类型、销毁方式，由墨子（后端）、皋陶（gāo yáo）共同确认。

## 六、责任与监督

- 墨子（后端）：负责数据存储、传输、销毁的技术实现，保障技术层面的数据安全。

- 皋陶（gāo yáo）：负责监督数据处理全流程，定期排查数据安全风险，审核数据收集、使用、销毁方案。

- 责任追究：如出现数据泄露、违规收集、违规使用等情况，追究相关负责人责任；情节严重的，按法律法规处理。

## 七、附则

1.  本指南由皋陶（gāo yáo）负责维护、更新，与国家法律法规冲突时，以国家现行法律为准。
