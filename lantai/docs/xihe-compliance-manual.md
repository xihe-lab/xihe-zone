# 羲和实验室合规手册

## 一、总则
1. 本手册为羲和实验室内容、视觉、宣传、项目、官网运营的唯一合规依据。
2. 所有文案、页面、图片、视频、文档上线前，必须经**皋陶（gāo yáo）**审核通过。
3. 合规底线：文化可用，宗教不可用；创意可行，红线不可碰。

## 二、适用范围
- 官网（xihe.zone）所有页面、文案、素材、宣传内容
- 智能体团队命名、介绍、形象展示
- 兰台文档中心发布的所有公开资料
- 对外宣传、介绍、PPT、视频等相关材料

## 三、核心合规红线（严禁触碰）
1. **禁止宗教相关内容**
   - 不出现宗教教义、仪式、祭祀、祈福、道场、法会、开光等表述
   - 不使用宗教符号、建筑、服饰、法器等视觉元素

2. **禁止神化、崇拜化、封建迷信内容**
   - 不将神话人物作为神明进行崇拜、供奉、仪式化描述
   - 不涉及算命、占卜、风水、改运、通灵等内容

3. **禁止违规文化与历史表述**
   - 不歪曲、恶搞、滥用历史与传统文化符号
   - 不使用敏感人物、敏感事件、不当隐喻

4. **禁止侵权行为**
   - 不盗用图片、字体、音乐、代码、文案等有版权的内容
   - 不冒用机构、企业、他人名义进行宣传

5. **禁止隐私与信息违规**
   - 不擅自收集、存储、使用、泄露用户个人信息
   - 不设置不必要的表单、登录、授权入口

6. **禁止虚假与夸大宣传**
   - 不虚构技术成果、项目进度、团队背景
   - 不进行超出能力范围的技术承诺

## 四、文化使用安全规范
### 允许使用
- 上古神话、历史人物名称（羲和、鲁班、墨子、祝融、白泽、皋陶（gāo yáo）等）
- 东方美学、古风科技、黑金风格、卷轴、线条等设计元素
- 文化意象、诗意表述、团队代号、实验室命名

### 禁止使用
- 神、神殿、天宫、道场、法会、祭祀、祈福、修行、渡劫等词汇
- 宗教风格建筑、仪式、神像、祭祀场景

## 五、网站运营合规
1. 网站定位为技术学习、AI 探索、项目展示、知识沉淀的非经营性平台。
2. 不开展金融、医疗、教育、电商等需专项许可的业务。

## 六、内容审核流程（皋陶（gāo yáo） 负责）
1. 新内容/新页面/新素材 → 提交皋陶（gāo yáo）审查
2. 对照本手册逐项核对风险点
3. 合规 → 通过上线
4. 存在风险 → 修改后重新审核
5. 违规 → 禁止上线并记录

## 七、附则
1. 本手册由**皋陶（gāo yáo）｜法律合规与风险审查**负责维护与更新。
2. 本手册内容与国家法律法规冲突时，以国家现行法律为准。
3. 本手册公开放置于兰台，接受全员查阅与监督。
