## 🚀 本地预览

### 方式一：直接打开 HTML

```bash
# 在浏览器中打开
open src/index.html
# 或
xdg-open src/index.html  # Linux
```

### 方式二：使用 Live Server

```bash
# 安装 live-server
npm install -g live-server

# 启动服务器
cd xihe-zone
live-server src

# 访问 http://localhost:8080
```

### 方式三：使用 VitePress（推荐）

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:5173
```

## ✨ 核心视觉效果

### 1. 太阳光晕效果
- **位置：** Hero 区域背景
- **效果：** 金色光晕缓慢脉动
- **动画：** 6 秒周期，不透明度 0.5 ↔ 0.8

### 2. 宫殿卡片
- **位置：** 所有内容卡片
- **效果：** 深色渐变背景 + 金色顶部光条
- **交互：** 悬停上浮 4px + 金色阴影

### 3. 金色渐变文字
- **位置：** 所有标题
- **效果：** 金色渐变填充 + 光泽流动
- **动画：** 3 秒周期，从左到右

### 4. 按钮光芒
- **位置：** 所有按钮
- **效果：** 悬停时圆形光晕从中心扩散
- **交互：** 点击波纹效果

### 5. 云纹分隔线
- **位置：** 区块之间
- **效果：** 金色渐变线条 + 两端云纹

### 6. 滚动进度条
- **位置：** 页面右侧
- **效果：** 金色渐变进度指示器
- **交互：** 随滚动实时更新
