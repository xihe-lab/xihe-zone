## 🚀 本地开发

### 方式一：直接用 Markdown 阅读器

任何 Markdown 编辑器都可以打开 `index.md` 预览。

### 方式二：用 VitePress 部署为网站

```bash
# 克隆仓库
git clone git@gitee.com:xihe-lab/xihe-zone.git

# 进入目录
cd xihe-zone

# 安装 VitePress
npm install -D vitepress

# 启动本地开发服务器
npx vitepress dev

# 访问 http://localhost:5173
```

### 方式三：部署到 Gitee Pages

1. 在 Gitee 仓库设置中启用 Pages 服务
2. 选择 `master` 分支
3. 绑定自定义域名 `xihe.zone`
